# Сборка контента для каталога

Этот файл нужен как рабочая схема для сборки материалов по сайту «Системы хранения металла».

## Роли

- Настя — основной сборщик контента.
- Дмитрий — консультант по источникам, названиям и корректности материалов.
- Михаил — владелец структуры сайта и финального отбора.

## Что собираем в первую очередь

1. Категории каталога.
2. Товары внутри категорий.
3. Фото товаров.
4. Краткие описания.
5. Технические характеристики, если они есть.
6. PDF, схемы, паспорта, если они есть.

Цены в первой волне не обязательны.

## Структура папок

Корневая папка:

`Каталог`

Пример:

```text
Каталог/
  01_Автоматизированные_системы_для_хранения_листового_металла/
    001_Название_товара/
      photo_01.jpg
      photo_02.jpg
      passport.pdf
      scheme.pdf
    002_Название_товара/
  02_Ручные_системы_для_хранения_листового_металла/
  03_Системы_для_хранения_сортового_и_трубного_металлопроката/
```

## Таблица по товарам

Одна строка = один товар.

Минимальные колонки:

- `category_number`
- `category_name`
- `product_number`
- `product_name`
- `short_description`
- `full_description`
- `application`
- `characteristics`
- `main_photo`
- `additional_photos`
- `pdf_files`
- `notes`
- `content_status`

## Правила

- Использовать только внутренние материалы компании.
- Не брать чужие тексты и фото из интернета без отдельного согласования.
- Если данных не хватает, помечать `нет данных` или `уточнить`.
- Названия файлов в таблице должны совпадать с реальными файлами в папках.
- Один товар не должен быть размазан по нескольким разным папкам.

## Что делаем после первой волны

- Выбираем приоритетные категории для публикации.
- Утверждаем шаблон карточки товара.
- Начинаем переносить реальные товары в сайт до подключения CMS.
