# CHANGELOG

## 2026-04-28 — Industrial MVP release

### Что изменилось

- Создан Next.js/React MVP сайта «Системы хранения металла» для КБ Парус.
- Добавлен промышленный fullscreen hero с pseudo-video визуалом.
- Подключён реальный логотип КБ Парус.
- Добавлены основные продуктовые блоки: проблемы склада, что можно хранить, решения, до/после, как это работает, карточки оборудования, проектирование, преимущества, сценарии применения, данные для инженера и FAQ.
- Калькулятор переделан в многошаговый конфигуратор с прогрессом, карточками выбора, live summary, предварительным диапазоном цены и отправкой заявки.
- Добавлен API `/api/leads` с mock-режимом и подготовкой под Bitrix24.
- Добавлена структура данных под CMS и 1С.
- Добавлены favicon, иконки `lucide-react`, папка для будущих изображений и визуальные placeholder-изображения.
- Проект задеплоен в существующий Vercel project `kbparus-metal-storage`.

### Что проверить

- `npm run lint`
- `npm run test`
- `npm run build`
- Главная страница открывается.
- В шапке и футере отображается логотип.
- На сайте нет публичного блока конкурентных референсов.
- Калькулятор проходит шаги и показывает предварительный диапазон.
- `/api/leads` отвечает `200` в mock-режиме.
- Production URL открывается: https://kbparus-metal-storage.vercel.app

### Известные ограничения

- GitHub repository не создан из текущей среды: нет `gh`, GitHub token или авторизованного remote.
- Визуалы пока временные и должны быть заменены на реальные фото/видео КБ Парус.
- Bitrix24 webhook не подключён в окружении production.
